# Frontend Portfolio

A modern portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

## Scripts

- `npm install` installs dependencies
- `npm run dev` starts the local development server
- `npm run build` creates a production build
- `npm run preview` previews the production build locally

## Notes

- Update the content in `src/data/portfolio.js` to personalize projects, links, stats, and contact details.
- The main UI structure lives in `src/App.jsx`.
- Global styling and the Tailwind theme live in `src/index.css`.
